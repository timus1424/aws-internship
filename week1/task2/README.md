## Key Takeaways

This task introduced the concept of Identity and Access Management (IAM) 
in AWS. I learned how groups simplify permission management, how custom 
JSON policies implement the Principle of Least Privilege, and how AWS 
evaluates permissions through explicit allows and implicit denies. 
Using the AWS CLI, I verified that a restricted IAM user could 
successfully perform only the actions permitted by the attached
policy while being denied unauthorized operations.
